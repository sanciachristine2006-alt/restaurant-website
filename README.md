# 🍽️ Restaurant Website

This is a simple restaurant website project built using HTML.  
It showcases a basic layout for a restaurant including menu, contact details, and design structure.

## 📌 Features
- Homepage with restaurant introduction
- Menu section
- Contact information
- Simple and clean design

## 🛠️ Tech Used
- HTML

## 📂 Project Structure
- restaurant-website.html → Main website file

## 🚀 How to Run
1. Download or clone the repository
2. Open `restaurant-website.html` in your browser
